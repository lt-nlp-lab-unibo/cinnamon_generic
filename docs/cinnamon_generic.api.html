Builds a Routine component and runs it in inference mode.

Parameters:

  • routine_path (Union[AnyStr, Path, None]) – path where Routine training result is stored

  • run_name (Optional[str]) – directory name under ‘pipelines’ folder where Routine training result is stored.

  • serialize (bool) – if True, it enables the serialization process of Routine component during execution.

  • namespace (Optional[str]) – registration key namespace

Raises

AttributeError: if both routine_path and routine_name are not specified. FileNotFoundError: if no train command metadata file is found.

cinnamon_generic.api.commands.setup_registry(directory=None, module_directories=None, registrations_to_file=False, file_manager_key=None)#

This command does the following actions: - Populates the Registry with specified registration actions. - Builds the FileManager Component and stores its instance in the Registry for quick use. - Set-ups the logging utility module. - If generate_registration, invokes the list_registrations command for debugging and readability purposes.

!IMPORTANT!: this command is always required at beginning of each of your scripts for proper Registry initialization.

Parameters:

  • directory (Union[AnyStr, Path, None]) – path to the base directory where to look for standard FileManager folders.

  • module_directories (Optional[List[Union[AnyStr, Path]]]) – list of base directories where to look for registration calls.

  • registrations_to_file (bool) – if True, the list_registrations command is invoked.

  • file_manager_key (Union[RegistrationKey, str, None]) –

Return type:

FileManager

Returns:

The built FileManager
